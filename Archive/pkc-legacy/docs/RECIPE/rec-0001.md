# Create VFX List from Premiere Markers

**ID:** `REC-0001`
**Kind:** `recipe`

## Purpose

Generate a VFX spreadsheet from locked Premiere edit markers.

## Uses

- OBJ-0004
- OBJ-0010

## Steps

- Export sequence XML.
- Parse marker names and timecodes.
- Use ffmpeg to extract thumbnails and previews.
- Generate XLSX with shot id, thumbnail, preview path, task and status.

## Validation

- Every marker must have unique CG shot id.
- Every preview file must exist.
