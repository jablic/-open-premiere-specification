# Developer API Inventory

This directory contains machine-readable inventory files for Premiere Pro developer surfaces.

The inventory is not prose documentation. It is the working index from which complete reference pages are planned and verified.

## Status values

- `official`
- `adobe_sample`
- `reproduced`
- `community_verified`
- `hypothesis`
- `unknown`

## Coverage workflow

1. Add symbol to inventory.
2. Attach evidence.
3. Create or link reference page.
4. Add behaviour contract if known.
5. Mark gaps explicitly.
